
# Watch Store E-commerce Website

A full-featured **Watch Store E-commerce Website** built with **Django**, where users can browse watches, view product details, manage their cart, and place orders seamlessly.

## Features

* User Registration & Login
* Product Listing
* Product Detail Page
* Shopping Cart
* Order Placement
* Admin Dashboard
* Responsive UI
* Category-wise Product Display

## Tech Stack

**Backend:**

* Python
* Django

**Frontend:**

* HTML
* CSS
* JavaScript
* Bootstrap

**Database:**

* SQLite

## Installation Guide

### 1. Clone the repository

```bash
git clone https://github.com/your-username/Watch-Store.git
cd Watch-Store
```

### 2. Create virtual environment

```bash
python -m venv env
```

### 3. Activate virtual environment

**Windows**

```bash
env\Scripts\activate
```

**Mac/Linux**

```bash
source env/bin/activate
```

### 4. Install dependencies




### 5. Run migrations

```bash
python manage.py migrate
```
